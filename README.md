## Padframe designs

The individual designs from different teams are here:

- M2 Chip Odyssey: TIA (bottom), Comparator and SAR logic (top) (5MHz clock). Repo [link](https://github.com/miakadam/Wk_Chip_Odyssey)
- M3 CreActive: Active Filter Building Blocks (6 OTAs, 6 Capacitors, 12 Resistors). Repo [link](https://github.com/assaify/creactive-chipathon-2025)
- M9 Prof Morbius: RF Mixer (250MHz max). Repo [link](google.com)
- M10 RAZAVUS. Repo [link](google.com)
- M13 SpikCore. Analog neurons(1 LIF, 5 AH); 6 synapses and 1 bootstrapped TG. Repo [link](https://github.com/RoyceRichmond/Mosbious_2025_Spikcore)
